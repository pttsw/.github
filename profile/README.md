# 影魔网图书馆

**The Dark Lady's Weave is the only true source of power. Servants of Mystra are restricted by their goddess, for Mystra has established limitations and the bounds of magic. Followers of the Dark Lady are subject to no such limitations.
”**
<br>***— Oruana Drearon, (incorrect) monk of the Dark Moon.t***

影魔网图书馆是一个专注建设**DND5 中文共库**（以下简称共库）的民间组织，致力于构建**英文数据库（魔网）的中文复制体（影魔网）**，供各个工具与文档站使用。

共库以[5etools](https://5e.tools/)的[仓库源码](https://github.com/5etools-mirror-3/5etools-src)为基础，在此基础上将英文替换成中文，然后将其输出成文档站或跑团工具所使用的 json 文件。

共库内容均为爱好者自发翻译整理，仅供学习、交流、参考使用，不用于任何商业用途。

共库中出现的公司名称、商标、产品等版权归版权方所有，出自5e tools的内容使用[MIT许可](https://zh.wikipedia.org/zh-cn/MIT%E8%A8%B1%E5%8F%AF%E8%AD%89)，出自不全书的内容使用[gpl3许可](https://choosealicense.com/licenses/gpl-3.0/)。若无另外声明，其余内容采用[知识共享署名-非商业性使用-相同方式共享](https://creativecommons.org/licenses/by-nc-sa/4.0/deed.zh-hans)授权，转载请注明译者与来源链接。
## 为何使用[5etools源码](https://5e.tools/)作为数据库英文源码？

- [5etools源码](https://github.com/5etools-mirror-3/5etools-src)完全开源，只要遵守[MIT](https://zh.wikipedia.org/zh-cn/MIT%E8%A8%B1%E5%8F%AF%E8%AD%89)协议皆可使用，且更新十分快速。
- [5etools源码](https://github.com/5etools-mirror-3/5etools-src)的有一套成熟的 json 数据，且兼容诸如 Foundry VTT、枭熊2、Roll20等跑团平台，十分适合作为数据库底层逻辑。

## 共库使用实例
当前[dnd5wiki](https://dnd.huijiwiki.com)与最新的[5etools中文站](https://5e.kiwee.top/)、[Plutonium-CN中文版](https://github.com/5eTools-Plutonium-CN/Plutonium-CN-Next/releases/)均使用共库作为自己的数据来源，这些工具都可在网络使用（请注意以上网站跟工具都跟共库一样处于内测阶段）。

## 我可以使用共库的数据文件吗？
当然可以，只要你遵守我们的协议。共库的内容与工具完全开源，你可以直接使用[5et汉化站的源码](https://github.com/pttsw/5etools-mirror-2.github.io)（源码的data文件里的json文件完全由共库输出而成），如果你使用时对json有长度限制要求，你可以使用[wiki-dnd-parser
脚本](https://github.com/pttsw/wiki-dnd-parser)来输出短的json文件。目前共库尚处于测试阶段，无法提供json文件的使用教程，若想使用可以加入共库群：[1045869232](https://qm.qq.com/q/xGtJLsk0tW)询问其他开发者的经验，或者自行摸索共库内容。

## 共库目前的问题
共库目前还处于测试阶段，因此目前还有许多问题：
- 人译内容稀缺，只有6成左右的内容是精翻，其他内容里有许多史前遗留下来的机翻文本（但机翻文本都有标注出来）。
- 其次是目前[共库编辑站](http://kiwee.top:11321/#/table/dashboard)还十分简陋，无法实现一些进阶功能。
- 目前共库教程指包括编辑站这块，在其他方面依然有所缺失。
